## Elixir types -- Collection types

|type|examples|
|------|------|
| Lists | `[ 1, 2, 3]` |
| Keyword Lists | `[ from: 'me', to: 'you' ]`, `[{:from, 'me'},{:to, 'you'}]` |
| Tuple | `{1, two, 3}`|
| Maps | `%{ a: 1, b: 2}`, `%{ :a => 1, :b => 2}` |
| Structs | ```defmodule TestStruct do``` <br> ```	defstruct val1: 0, val2: 1``` <br> ``` end``` |
